Commands  
--------
> npm uninstall -g @angular/cli		// will install latest NG CLI
> npm install -g @angular/cli@6
> npm install bootstrap@3 --save	// bootstrap CSS library
> npm install rxjs-compat --save 	// rxjs 5 to rxjs 6 compatability
> npm cache clean --force		// cleans the cache
> npm install firebase --save		// firebase SDK 
> npm install serve -g			// servers html file, zero config server


> ng -v
> ng new users-app (creates new angular project)
> cd users-app
> ng serve (starting webpack live dev server)
> ng generate component <COMP_NAME>
> ng generate directive <DIR_NAME>
> ng generate pipe <PIPE_NAME>
> ng build --prod --aot
> ng build --prod --aot --base-href="users-app"
